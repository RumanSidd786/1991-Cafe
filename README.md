1991 Cafe & Lounge Website

A premium and modern restaurant landing page for 1991 Cafe & Lounge, Lucknow built using HTML, Tailwind CSS, and Vanilla JavaScript.
This website showcases the cafe’s ambience, menu, gallery, customer reviews, and reservation/contact options in a stylish and responsive design.

✨ Features
🎨 Modern luxury UI design
📱 Fully responsive for mobile, tablet, and desktop
🍽️ Interactive menu section with category tabs
🖼️ Animated gallery section
⭐ Customer testimonials section
📍 Google Maps integration
📞 Direct call and WhatsApp reservation buttons
⚡ Smooth scrolling and animations
🌙 Elegant dark theme with gold accents
🛠️ Technologies Used
HTML5
Tailwind CSS
Vanilla JavaScript
Google Fonts
Google Maps Embed API
